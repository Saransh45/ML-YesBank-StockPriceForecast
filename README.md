# ML-YesBank-StockPriceForecast
This project demonstrates a machine learning solution for predicting Yes Bank stock closing prices. It uses regression models like Random Forest and XGBoost on historical stock data to forecast future closing prices. The project includes data preprocessing, feature engineering, model training, evaluation, and visualization of predictions.
